# 分数映射方案分析

## ASAP数据集分数分布

| Essay Set | 原始范围 | 样本数 | 特点 |
|-----------|---------|--------|------|
| Set 2 | 1-6 | ~1,800 | 范围最大，包含6分 |
| Set 3 | 0-3 | ~1,726 | 范围最小 |
| Set 4 | 0-3 | ~1,772 | 范围最小 |
| Set 5 | 0-4 | ~1,805 | 中等范围 |
| Set 6 | 0-4 | ~1,800 | 中等范围 |

## 方案对比

### 方案1: 统一到 0-5 分区间 ⭐ **推荐**

**优点**:
- 可以表示"极差"答案（0分）
- 与1-5分接近，最终输出只需+1
- 更好地利用所有数据范围
- 训练更稳定（0-5比1-5范围更宽）

**映射方式**:
- Set 2 (1-6): `(score - 1) * 5/5 = score - 1` → 0-5
- Set 3-4 (0-3): `score * 5/3` → 0-5
- Set 5-6 (0-4): `score * 5/4` → 0-5

**最终输出**: `final_score = model_score + 1` (转换为1-6，或直接使用0-5)

### 方案2: 统一到 0-4 分区间

**优点**:
- 更接近Set 5-6的原始范围
- 数据分布更均匀

**缺点**:
- Set 2的1-6范围压缩较多，可能丢失信息
- 与1-5分差距较大

### 方案3: 保持原始分数训练

**优点**:
- 不丢失信息
- 最灵活

**缺点**:
- 不同set的分数含义不同，模型需要学习多个分布
- 训练可能不稳定

## 推荐方案: 0-5分区间

**理由**:
1. **更好的数据利用**: Set 2的1-6可以映射为0-5（1→0, 6→5）
2. **包含极值**: 0分可以表示"完全错误"的答案
3. **易于转换**: 最终输出时 `+1` 即可得到1-6分，或直接使用0-5
4. **训练稳定**: 0-5范围比1-5更宽，梯度更稳定

**实现**:
```python
# 训练时: 0-5分
# 预测时: 可以选择
#   - 直接输出0-5
#   - 或转换为1-6: score + 1
#   - 或转换为1-5: score * 4/5 + 1
```
